# demo_vggt_3d

VGGT Space source with Colab demo and a multi-view generator via fal.

## Colab

Open `colab_vggt_demo.ipynb` and run the cells.

## Multi-view generation

Configure `multiview_config.yaml` and run:

```bash
python multiview_generate.py --config multiview_config.yaml
```

Check out the configuration reference at https://huggingface.co/docs/hub/spaces-config-reference
